# Mock Login Page

This page provides predefined test credentials for easy testing of the Vivaha wedding planning application.

## Access

Navigate to `/mock-login` or click the "Mock Login" button in the header.

## Features

- **Quick Login**: Click "Use" button to auto-fill credentials
- **Copy to Clipboard**: Click the copy icon to copy credentials
- **Auto-Create Users**: The system automatically creates users on first login
- **Persistent Sessions**: Login state is maintained across browser sessions

## Usage

1. Navigate to the mock login page
2. Either:
   - Click "Use" on any credential to auto-fill the form
   - Manually enter email and password
   - Copy credentials to clipboard for external use
3. Click "Sign In" to authenticate
4. You'll be redirected to the chat interface

## Notes

- These are mock credentials for testing purposes only
- Any email/password combination will work (auto-creates users)
- Sessions persist until manually logged out
- All test data is stored in memory and resets on server restart
